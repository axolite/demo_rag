# Pinned Python dependency lockfile for the NCS v1.6.1 doc build (two-phase repro).
#
# Ideally this is the exact pip-freeze of the first good build (see below). Until
# then it carries the MINIMAL interim caps the 2021 doc toolchain needs to build

# Jinja2/MarkupSafe deps loose, and unconstrained they resolve to current releases
# that dropped APIs Sphinx 3.5 imports (jinja2.environmentfilter removed in 3.1;
# markupsafe.soft_unicode removed in 2.1). Cap them to the 3.0/2.0 line that
# matches the mid-2021 NCS v1.6.1 build.
#
# After the first successful build, replace this whole file with the captured
# freeze to make every later build byte-stable:
#
#     cp /c/ncs-docbuild/out/pip-freeze.txt docker/constraints.txt
#
